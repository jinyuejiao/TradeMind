# 手机端展示优化计划

## 问题分析

根据用户需求，需要对手机端界面进行以下优化：

1. **手机端头部右上角增加用户图标**：
   - 当前手机端头部只显示品牌标识和搜索框（隐藏），没有用户图标
   - 需要添加用户图标，点击后弹出会员订阅和推荐信息
   - 与网页版用户图标点击流程保持一致

2. **手机端底部模块切换栏移除重复功能图标**：
   - 当前底部导航栏在[客户]和[产研]中间有一个扫码/AI按钮
   - 这个按钮与其他功能重复，需要移除
   - 移除后需要调整导航栏布局，使其他按钮均匀分布

## 修复方案

### 1. 在手机端头部添加用户图标
- 在header的右侧添加一个用户图标按钮
- 点击该按钮调用openMemberModal()函数，与网页版保持一致
- 确保在手机端显示，在桌面端隐藏

### 2. 移除手机端底部导航栏的重复功能图标
- 移除[客户]和[产研]中间的扫码/AI按钮
- 调整底部导航栏的布局，使其他按钮均匀分布
- 确保导航栏在手机端正常显示

## 修复步骤

1. **在手机端头部添加用户图标**
   - 在header的右侧添加一个用户图标按钮
   - 设置点击事件为openMemberModal()
   - 添加适当的样式，确保在手机端显示，在桌面端隐藏

2. **移除手机端底部导航栏的重复功能图标**
   - 移除[客户]和[产研]中间的扫码/AI按钮
   - 调整底部导航栏的布局，使其他按钮均匀分布
   - 测试导航栏的显示效果

3. **测试修复效果**
   - 测试手机端头部用户图标的显示和点击功能
   - 测试手机端底部导航栏的布局和功能
   - 验证所有模块切换功能正常

## 预期结果

- 手机端头部右上角显示用户图标，点击后弹出会员订阅和推荐信息
- 手机端底部导航栏移除了重复的扫码/AI按钮，其他按钮均匀分布
- 所有模块切换功能正常
- 界面布局美观，用户体验良好

- **低风险**：所有修改都在现有的代码结构基础上进行，不会破坏现有功能
