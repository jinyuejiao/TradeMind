# 供应商编辑弹窗与仓库管理抽屉 UI 实现计划

## 问题分析

根据用户需求，需要实现以下功能：

1. **供应商编辑弹窗**：
   - 在 `supply-chain.html` 中实现
   - 字段映射：供应商名称 ⬅️ name, 联系人 ⬅️ contact, 电话 ⬅️ phone, 评分 ⬅️ rating
   - 视觉规范：圆角 `rounded-[2.5rem]`，背景 `backdrop-blur-sm`

2. **仓库管理抽屉**：
   - 在 `product-center.html` 右上角新增 `ph-buildings` 图标按钮
   - 实现抽屉逻辑：点击后从右侧滑出宽度为 `w-80` 或 `w-96` 的面板
   - 内容展示：仓库列表，支持新增、修改仓库名称和位置
   - 风格对齐：抽屉背景使用 `bg-slate-900/90` 配合 `text-white`，与侧边栏风格保持一致

## 修复方案

- 实现供应商信息的编辑和保存功能

### 2. 仓库管理抽屉

- 实现抽屉的打开和关闭逻辑
