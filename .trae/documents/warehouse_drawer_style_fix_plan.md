# 仓库管理抽屉风格统一计划

## 问题分析

根据用户需求，仓库管理抽屉的界面风格需要与其他弹窗界面风格保持一致。通过分析当前代码，发现以下风格差异：

1. **颜色方案**：
   - 其他弹窗（如进货建议弹窗）：白色背景，深色文字
   - 仓库管理抽屉：深色背景（bg-slate-900/90），白色文字

2. **布局结构**：
   - 其他弹窗：居中弹窗，有最大宽度限制
   - 仓库管理抽屉：从右侧滑出的抽屉，固定宽度

3. **视觉元素**：
   - 其他弹窗：品牌色图标背景，白色文字
   - 仓库管理抽屉：品牌色图标背景，白色文字（这部分一致）

## 修复方案

### 1. 修改仓库管理抽屉的颜色方案
- 将背景颜色改为白色
- 将文字颜色改为深色
- 保持品牌色按钮的风格

### 2. 保持抽屉式布局
- 保留从右侧滑出的抽屉式布局，因为这是仓库管理功能的特色
- 调整抽屉的宽度和样式，使其与整体风格协调

### 3. 统一视觉元素
- 统一标题栏、内容区、按钮的样式
- 确保字体大小、间距、圆角等视觉元素与其他弹窗保持一致

## 修复步骤

1. **修改仓库管理抽屉的HTML结构**
   - 将背景颜色改为白色
   - 将文字颜色改为深色
   - 调整边框和分隔线的颜色

2. **修改CSS样式**
   - 确保输入框、按钮等元素的样式与其他弹窗保持一致
   - 调整间距和布局

3. **测试修复效果**
   - 打开仓库管理抽屉，验证风格是否与其他弹窗一致
   - 测试功能是否正常

## 预期结果

- 仓库管理抽屉的界面风格与其他弹窗保持一致
- 功能正常，用户体验良好
- 视觉效果统一，符合整体设计风格

## 风险评估

- **低风险**：修改主要是样式调整，不会影响功能
- **低风险**：所有修改都在现有的代码结构基础上进行
- **低风险**：风格调整不会破坏现有功能
