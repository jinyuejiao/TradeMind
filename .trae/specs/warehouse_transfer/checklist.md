# 仓库调拨 (Warehouse Transfer) - Verification Checklist

## 入口按钮验证
- [ ] 每个仓库的操作列中都有调拨按钮
- [ ] 调拨按钮位置在编辑图标左侧
- [ ] 按钮使用 ph-bold ph-swap 图标
- [ ] 按钮颜色是 text-amber-500
- [ ] 鼠标悬停显示"仓库调拨"提示（title属性）
- [ ] 点击按钮可以打开调拨弹窗

## 弹窗 UI 验证
- [ ] 弹窗尺寸为 max-w-4xl（PC端）
- [ ] 弹窗圆角为 rounded-[2.5rem]（PC端）
- [ ] 弹窗标题显示"仓库产品调拨"
- [ ] 弹窗头部正确显示源仓库名称
- [ ] 有关闭按钮（ph-x 图标）
- [ ] 点击背景遮罩可以关闭弹窗

## 目标仓库选择验证
- [ ] 有目标仓库选择下拉框
- [ ] 下拉框样式为 bg-slate-50
- [ ] 下拉框不包含当前源仓库
- [ ] 下拉框可以正常选择其他仓库

## 调拨类型选择验证
- [ ] 有"同价调拨"和"变价调拨"两个选项
- [ ] 使用药丸组（Pills）样式
- [ ] 默认选中"同价调拨"
- [ ] "同价调拨"选中时有白色背景和 teal-600 文字
- [ ] 切换选项时有视觉反馈
- [ ] 同价调拨模式下，单价和总价输入框被禁用（disabled）
- [ ] 变价调拨模式下，单价和总价输入框可以编辑

## 产品清单验证
- [ ] 表格包含产品名称列
- [ ] 表格包含单价列（输入框）
- [ ] 表格包含数量列
- [ ] 数量显示为"X箱X包"格式（或类似商贸格式）
- [ ] 表格包含总价列
- [ ] 表格样式与现有系统一致
- [ ] 表格有边框分隔（border-b border-slate-50）

## 实时价格计算验证
- [ ] 修改单价时，该行总价实时更新
- [ ] 任何价格变化时，底部调拨总价值实时更新
- [ ] 价格计算正确（单价 × 数量 = 总价）
- [ ] 总价显示格式正确（带货币符号，保留两位小数）

## 确认调拨验证
- [ ] 未选择目标仓库时有提示
- [ ] 有确认调拨按钮
- [ ] 点击确认后，控制台打印完整的调拨数据包（JSON格式）
- [ ] 调用 refreshData() 刷新界面
- [ ] 显示成功提示（toast消息）
- [ ] 操作完成后正确关闭弹窗

## 移动端适配验证
- [ ] 屏幕宽度 < 768px 时，弹窗全屏显示
- [ ] 移动端弹窗使用 fixed inset-0 rounded-none 样式
- [ ] 产品清单在移动端转为卡片布局
- [ ] 所有按钮大小适中，易于点击
- [ ] 间距在移动端合适，不拥挤

## 整体风格验证
- [ ] UI 风格与现有系统完全一致
- [ ] 使用相同的圆角、阴影、配色方案
- [ ] 图标风格一致（Phosphor Icons）
- [ ] 过渡动画流畅自然
- [ ] 所有交互有适当的 hover 效果
